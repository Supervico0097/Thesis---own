Please see the isntructions how to run the programs:

Datasets Generator:

1) Open config_music.yaml file
2) adjust the parameters to your prefernce
3) run data_generator.py

*******************

To test the colaborative filtering recommnder:

1) Open colaborative_filtering.py
2) adjust the parameters num_recommendations, num_neighbors, size of users list
3) run the file

*******************

To test the predictor:

1) Open "notebooks" folder
2) Open and run "Generate_Training_Data.ipynb"
3) Open "ml.ipyb"
4) set weeks number in second row based on actual number of weeks generated for the datasets
5) run "ml.ipyb"
